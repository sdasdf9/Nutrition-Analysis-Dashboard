# Пояснительная записка к проекту: «Анализ пищевой ценности продуктов»

## 1. Введение
Целью данного проекта является разработка информационно-аналитической системы (дашборда) для учета и анализа нутриентного состава различных продуктов питания. Приложение позволяет интерактивно оценивать долю продукта в суточной норме потребления и формировать статистические отчеты по макронутриентам (БЖУ).

## 2. Описание архитектуры и базы данных (PostgreSQL)
В качестве СУБД используется PostgreSQL. Приложение реализовано на Python (фреймворк Flask) по архитектурному паттерну MVC (Model-View-Controller). 
База данных `nutrition_db` спроектирована с учетом нормализации и состоит из двух основных связанных таблиц:

1. **Таблица `categories` (Категории продуктов):**
   * `id` (Primary Key) — уникальный идентификатор категории.
   * `name` (String) — наименование (например: "Мясо и птица", "Овощи", "Крупы").

2. **Таблица `products` (Продукты):**
   * `id` (Primary Key) — уникальный идентификатор.
   * `category_id` (Foreign Key) — связь с таблицей `categories` (отношение «Один ко многим»).
   * `name` (String) — название конкретного продукта.
   * `calories` (Float) — энергетическая ценность (ккал на 100 г).
   * `protein` (Float) — содержание белков (г).
   * `fat` (Float) — содержание жиров (г).
   * `carbs` (Float) — содержание углеводов (г).

## 3. Выводы по статистике
В ходе работы серверного скрипта данные из БД выгружаются в библиотеку `Pandas`, где проводится расчет агрегированных метрик (среднее, медиана, поиск экстремумов). 

**Основные выводы из анализа данных:**
* **Лидеры по белку:** Наибольшая концентрация белка ожидаемо наблюдается в категории мясных продуктов, морепродуктов и бобовых. Эти продукты формируют основу структурного профиля питания.
* **Энергетическая плотность (Калории):** Абсолютными рекордсменами по калорийности выступают орехи и масла (за счет высокой доли жиров, где 1 г жира = 9 ккал).
* **Медианные значения:** Сравнение среднего арифметического и медианы показывает, что по ряду категорий данные распределены неравномерно, присутствуют продукты-выбросы с аномально высокой пищевой ценностью.
