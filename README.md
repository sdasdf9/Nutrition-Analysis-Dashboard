# Анализатор пищевой ценности продуктов (Nutrition Analysis Dashboard)

Интерактивное веб-приложение для анализа состава БЖУ (белки, жиры, углеводы) и калорийности продуктов питания.

## Стек технологий
 Backend Python 3, Flask, SQLAlchemy
 Database PostgreSQL (запуск через Docker)
 Data Science Pandas (обработка данных), Matplotlib (визуализация)
 Frontend HTML5, CSS3 (CSS Grid, Flexbox), Jinja2

## Основной функционал
1. Интерактивный анализатор Поиск продукта по базе данных и визуализация процента от суточной нормы потребления (2000 ккал, 100г белка и т.д.) с помощью динамических шкал.
2. Аналитический дашборд Автоматический расчет статистических показателей (среднее, медиана, абсолютные минимумы и максимумы).
3. Генерация отчетов Построение столбчатых графиков распределения макронутриентов по категориям продуктов по запросу пользователя.
