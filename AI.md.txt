## Использование AI-ассистента при разработке

В процессе работы над проектом нейросеть (Gemini) использовалась в качестве справочника (вместо чтения документации) для решения локальных проблем с синтаксисом и отладки ошибок. 

**Примеры запросов (промптов), которые я использовал:**

1. **Поиск SQL-команд:**
   > *"Как написать SQL запрос в PostgreSQL, чтобы объединить таблицы products и categories по ID, если мне нужны только колонки name и calories?"*
   > *(Использовано для корректного написания конструкции JOIN).*

2. **Работа с Pandas:**
   > *"У меня есть датафрейм df. Как с помощью pandas найти строку с максимальным значением в колонке 'protein' и вывести название этого продукта?"*
   > *(Использовано для поиска продуктов-рекордсменов).*

3. **Отладка ошибок (Debugging):**
   > *"Почему Flask выдает ошибку 404 (Not Found) на файл style.css, хотя он лежит в папке? Как правильно подключить статические файлы в HTML шаблоне?"*
   > *(ИИ подсказал использовать функцию `url_for('static', filename='...')` вместо обычного пути).*

4. **Интеграция Matplotlib и Flask:**
   > *"Как сохранить график matplotlib не в картинку на жесткий диск, а в строку base64, чтобы передать её напрямую в HTML тег <img>?"*
   > *(Использовано для оптимизации работы с графиками).*
